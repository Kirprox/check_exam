# CheckExam Service

Сервис checkexam предназначен для хранения результатов пользователей по экзаменам.

Сервис реализован на Spring Boot и является частью микросервисной архитектуры, развернутой в Kubernetes.

### Назначение сервиса

Сервис хранит информацию о том, какой пользователь сдавал какой экзамен.

#### Каждая запись содержит:

id записи

clientId — идентификатор пользователя

descId — идентификатор экзамена

Основная логика сервиса

Сервис не хранит название экзамена, только его идентификатор.

Для получения названия экзамена сервис обращается к checkdesc.

Таким образом происходит обогащение данных.
